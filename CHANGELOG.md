# Change Log

## v1.1.27 → v1.2.0

- Added:
  - **Try/catch incentive.** `.then()` and `.catch()` usage is prohibited. Using try/catch instead is encouraged.
  - **`NeatlintOptions.append` field.** Append new rules to the ESLint settings within Neatlint.

## v1.0.4 → v1.1.0

- Added:
  - **`NeatlintOptions.disabled` field.** ESLint completely frees up your settings.
  - **More rules.** Rules that prioritize type safety and logic errors.
